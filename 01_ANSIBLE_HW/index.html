01 ANSIBLE

Run it

ansible-playbook playbook.yml --vault-password-file vault_pass

Homework aliases

  • Create a inventory file with four groups, each provisioning VM should be in separate group and group named iaas what should include childrens from two first groups; hosts.yml
  • Create reusable roles for that:
  • creating a empty file /etc/iaac with rigths 0500, this role should be idempodent 01_iaac
  • define content of /etc/issue as variable 02_issue
  • Create playbook for: playbook.yml
  • invoke the role for /etc/iaac for hosts group iaas
  • invoke the role for defining variable for all hosts
  • print in registered variables
  • printing hostnames together with registered variables will be a plus.
  • Create a repo in your GitHub account and commit code above repo

Optional:

  • use ansible_user and ansible_password for ssh connection and store passwords for each VM in encrypred way (add vault password to README.md in this case) host_vars (ip, user, sudo password) encrypted with password
